# 最长上升子序列

## 常规做法

dp[i]表示以第i个数为结尾的最长上升子序列的长度。

$$dp[i] = max(dp[j]) + 1, (a[j] < a[i] \&\& j < i)$$

复杂度$$O(n^2)$$。

## 优化

### 优化方法1

dp[i]表示长度为i的最长上升子序列的末尾最小值。
